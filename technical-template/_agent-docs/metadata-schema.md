## Naming Conventions

### Files
- **Daily notes:** `YYYY-MM-DD`
- **Weekly notes:** `YYYY-WW`
- **Monthly notes:** `YYYY-MM`
- **Media entries:** Title of work
- **Recipes:** Recipe name

### Properties
- **PascalCase:** `Category`, `Created`, `Modified`, `References`, `Links`
- **lowercase:** `rating`, `status`, `tags`, `url`, `type`
- **kebab-case:** `hours-played`, `book-reading`, `3d-printing`
- **Plugin prefixes:** `excalidraw-*`, `TQ_*`

### Tags
- **Hierarchical:** `parent/child/grandchild`
- **Genre tags:** `Movies/Genre/Action`, `TV/Genre/Comedy`
- **Type tags:** `note/info`, `note/knowledge`

## Property Patterns by Template

### Periodic Notes
```yaml
type: daily|weekly|monthly
tags: [daily|weekly|monthly]
date: YYYY-MM-DD
week: YYYY-WW
month: YYYY-MM
banner: "url"
banner_y: 0.5
cssclasses: [hide-properties]
```

### Media Templates
```yaml
category: books|video-games|TV|Movies
genre: [wikilinks]
year: YYYY
rating: 0-10
status: [playing|completed|dropped]
posterurl: "url"
tags: [type-specific]
cssClass: cards
```

### Product/App Templates
```yaml
Created: YYYY-MM-DD
Modified: YYYY-MM-DD
Category: "Apps"|"Phone-Apps"|"Subscription"
type: [app type]
os: [windows, mac, linux] | [android, iOS]
installed: true|false
usage: 0-4
```

## Meta Bind Field Patterns

### Toggle Switches
```
INPUT[toggle:propertyName]
```

### Progress Bars
```
INPUT[progressBar(minValue(0), maxValue(10), stepSize(1)):propertyName]
```

### Sliders with Custom Range
```
INPUT[progressBar(minValue(0), maxValue(360), stepSize(30)):timeProperty]
```
